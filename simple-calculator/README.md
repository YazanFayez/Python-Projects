# Simple Python Calculator

A robust command-line calculator built with Python. It features a continuous loop, error handling, and a colorful interface.

## Features
- **Basic Math:** Addition (+), Subtraction (-), Multiplication (*), Division (/).
- **Advanced Math:** Modulus/Remainder (%) support.
- **Safety:** Prevents crashing on "Division by Zero" or invalid text inputs.
- **Visuals:** Uses `colorama` for red error messages.
- **User Friendly:** Continuous loop until you type 'q' to exit.

## How to Run
1. Ensure you have Python installed.
2. Install the required library:
   ```bash
   pip install colorama
